# pt2-clone

This is a personal fork of [8bitbubsy/pt2-clone](https://github.com/8bitbubsy/pt2-clone).

Current version: `v1.91c`

# New Keybinds

- <kbd>Esc</kbd> is now for toggling edit mode. If the song is playing, pressing <kbd>Esc</kbd> will pause the song without entering edit mode.
- <kbd>Space</kbd> is now for toggling playback. If the song is playing, pressing <kbd>Space</kbd> will pause the song instead of re-playing it.
- It is now more intuitive to switch sample using the keypad:
    - <kbd>0</kbd> ~ <kbd>9</kbd> give 0 ~ 9
    - <kbd>/</kbd> gives A
    - <kbd>\*</kbd> gives B
    - <kbd>-</kbd> gives C
    - <kbd>+</kbd> gives D
    - <kbd>Enter</kbd> gives E
    - <kbd>.</kbd> gives F
    - Achieve 10 ~ 1F by holding <kbd>Shift</kbd>.
- <kbd>Shift</kbd>+<kbd>Del</kbd> to kill the sample.

# New Features

- Pattern row indices are displayed as HEX instead of DEC.
