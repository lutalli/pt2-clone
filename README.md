# pt2-clone

This is a personal fork of [8bitbubsy/pt2-clone](https://github.com/8bitbubsy/pt2-clone).

Current version: `1.91a`

# New Keybinds

- <kbd>Esc</kbd> is now for toggling edit mode. If the song is playing, pressing <kbd>Esc</kbd> will pause the song without entering edit mode.
- <kbd>Space</kbd> is now for toggling playback. If the song is playing, pressing <kbd>Space</kbd> will pause the song instead of re-playing it.
